# ISO 27001 Compliance

EAOS ISO 27001 compliance engine for Information Security Management.

## Overview

ISO 27001 provides requirements for establishing, implementing, maintaining, and continually improving an information security management system (ISMS).

## Running ISO 27001 Checks

```bash
npx eaos compliance iso27001
```
