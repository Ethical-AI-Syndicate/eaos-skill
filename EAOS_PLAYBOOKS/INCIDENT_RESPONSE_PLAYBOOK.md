## 2. Triage

## 3. Containment
- Isolate affected services
- Disable impacted credentials or endpoints
- Apply firewall or routing rules

## 4. Eradication
- Patch vulnerabilities
- Remove malicious artifacts
- Reset compromised credentials

## 5. Recovery
- Restore services
- Validate integrity
- Run post-recovery testing

## 6. Lessons Learned
- Root cause analysis
- Update runbooks
- Generate BEADS for follow-up
